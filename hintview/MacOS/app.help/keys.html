Key Strokes

Hintview recognizes the following key strokes. Most of these actions can also be invoked from the Menu.

  • ⌘A
    Toggle autoreload mode.

    See also the ⌘R key and the Preferences.

  • ⌘F
    Toggle showing the search text field.

    See also the Preferences.

  • ⌘G
    Find the next occurence of the search text.

  • ⇧⌘G
    Find the previous occurence of the search text.

  • Home, ⌘T
    Move to the "home" page. The home page is a location specified by the document author as a good starting point for reading the document.

    See also the Preferences.

  • ⌘N
    Toggle between night and day mode.

  • ⌘O
    Open a HINT file for viewing.

    See also the Preferences.

  • ⌘S
    Show a clickable outline of the content if the document provides it. 

  • Page-Down
    Move to the next page.

  • Page-Up
    Move to the previous page.

  • ⌘Q
    Quit Hintview.

  • ⌘R
    Reload the content. The file is reloaded and the viewer attempts to position the new file at about the same position as the old file. This attempt might fail if large scale changes have been made to the file.

    See also the ⌘A key.

  • ⌘Z
    Zoom to 100%.

    See also the Mouse Actions

  • ⌘,
    Show the preferences pane.

  • ?
    Show Help on using Hintview.
